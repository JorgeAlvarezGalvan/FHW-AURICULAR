# Auriculares

## 1. Definición del periférico

Un **auricular** es un periférico de salida de audio que permite escuchar sonidos generados por un dispositivo electrónico como ordenadores, teléfonos móviles, tablets o consolas. Algunos auriculares incorporan micrófono, por lo que también pueden funcionar como periféricos de entrada.

![Auriculares](images/auricular.jpg)

---

## 2. Características principales

- Permiten la reproducción de sonido de forma privada
- Pueden incluir micrófono integrado
- Conexión por cable o inalámbrica
- Diferentes niveles de calidad de audio
- Uso personal y profesional

---

## 3. Tipos de auriculares

- **Auriculares de diadema**: cubren completamente o parcialmente las orejas.
- **Auriculares intraurales**: se colocan dentro del oído.
- **Auriculares supraaurales**: apoyan sobre la oreja.
- **Auriculares con cable**: conexión mediante jack o USB.
- **Auriculares inalámbricos**: conexión por Bluetooth.

---

## 5. Conclusiones

Los auriculares son un periférico esencial en la actualidad, tanto para el ocio como para el trabajo. Su variedad de tipos permite adaptarse a diferentes necesidades como videojuegos, llamadas, estudios o entretenimiento. Gracias a los avances tecnológicos, ofrecen cada vez mejor calidad de sonido y mayor comodidad.
